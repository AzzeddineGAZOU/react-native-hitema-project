#Contexte du projet : Application de vente simple

##Description : 

 - La base de donnée est hébergée chez firebase
 - L'applicaiton gère l'authentification des users 
 
##Principe : 

 - L'application simule le système de vente en ligne sans l'option de paiement
 - L'utilisateur peut créer un article pour le mettre en vente, commander et acheter des articles
 - Modifier ou supprimer ses articles ( sans pouvoir toucher ceux des autres)
 
##Axes d'amélioration :
 - Recupération de mot de passe par email
 - Confirmation lors de la création d'un compte
 - Ajouter un filtre pour filtrer les articles

 ##Exécution de l'application:
 -npm run initialize
 -npm run start

    remarque : passer en connexion tunnel sur expo, si la compilation tarde
